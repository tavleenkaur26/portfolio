# Portfolio Website
My personal portfolio site, built to showcase my projects, experience, and what I'm currently working on.

Live at: https://tavleenkaur.vercel.app

## Built with

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Running locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
